Projects

Selected work

Hierarchical Bayesian Mixture Model — Grocery Shopping Segments

A PyMC model that uncovers latent household shopping segments from the Dunnhumby "Complete Journey" grocery transaction dataset. Incorporates product-level covariates — price, discounts, and popularity — to distinguish consumer-driven from retailer-influenced purchase behaviour through a hierarchical Bayesian mixture framework.

PyMC Bayesian Inference Python Mixture Models

Organisation Frequency Merger Toolkit

A pipeline for merging organisation mentions that appear under different names — Indonesian versus English variants, acronyms, formatting differences — into canonical forms for accurate frequency counting and network analysis. Uses Ollama as the LLM router to intelligently group bilingual organisation name variants.

LangChain Ollama NLP Python Entity Resolution

Pokémon Regression Modelling

An exploratory data analysis using the Pokémon dataset from the pokemon R library, modelling how speed points vary by primary type, secondary type, and weight through regression analysis and visualisation.

R ggplot2 Regression Statistical Modelling
